북마클릿 기능으로 간단하게 크랙 이삿짐을 싸봅시다! SillyTavern용 채팅 파일을 바로 저장할 수 있어요.

🤨북마클릿이란?

북마클릿 복사

PC에서는 이 버튼을 북마크바로 드래그

사용 방법

  1. 위의 복사하기 버튼을 누릅니다.
  2. 인터넷 브라우저 앱에서 아무 페이지나 열고 새 북마크를 만들어줍니다.
  3. 북마크 목록에서 방금 만든 북마크를 찾고 수정 버튼을 누릅니다.
  4. 이름 칸에는 알아보기 쉬운 새 이름을, URL 칸에는 방금 복사한 코드를 붙여넣습니다.
  5. 이삿짐을 싸고 싶은 채팅 페이지로 들어가서 맨 첫번째 채팅까지 스크롤을 올려 모두 로딩합니다.
  6. 그 페이지에서 아까 저장한 북마크를 바로 실행합니다.

    모바일 Chrome은 주소창에 북마크 이름을 입력한 뒤 검색 결과에 뜨는 북마크를 선택하는 방식이 더 잘 됩니다. iPhone에서는 Safari에서 먼저 테스트하는 편이 안정적입니다.

  7. 유저 이름, 캐릭터 이름, 저장 파일명을 차례대로 입력합니다.

    유저 이름과 캐릭터 이름은 채팅 창에서 메시지를 보낸 사람의 이름이 됩니다.

  8. 잠시 기다리면 .jsonl 파일이 다운로드됩니다!

    jsonl 파일은 실리태번이 채팅을 저장하는 파일 형식이에요. 해당 파일을 채팅으로 불러오면 이사 완료!
